# `secretclaw pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/start/pairing)

## Commands

```bash
secretclaw pairing list whatsapp
secretclaw pairing approve whatsapp <code> --notify
```
