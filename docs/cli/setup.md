# `secretclaw setup`

Initialize `~/.secretclaw/secretclaw.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
secretclaw setup
secretclaw setup --workspace ~/.secretclaw/workspace
```

To run the wizard via setup:

```bash
secretclaw setup --wizard
```
